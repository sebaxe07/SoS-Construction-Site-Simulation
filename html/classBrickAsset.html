Detailed Description

Behavior Tree Scriptable Object. BrickAssets can be attached to GameObjects or not.

You can create a Instance of a BrickAsset using the context menu (Right Click in Project View -> Create -> Behavior Tree) or with the New Tab Icon in the BB Editor.

Add it to a BehaviorExecutor to execute its internal behavior.
