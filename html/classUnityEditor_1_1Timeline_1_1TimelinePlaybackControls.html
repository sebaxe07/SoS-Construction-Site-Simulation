enum UnityEditor.Timeline.TimelinePlaybackControls.Context

Use Context to specify whether the time is based on local time or global time.

Enumerator
Local 

Time is relative to the current Timeline.

Global

◆ GetCurrentFrame()

int UnityEditor.Timeline.TimelinePlaybackControls.GetCurrentFrame ( Context context = Context::Local)
inline

Retrieves the location of the timeline playhead in frames.

Parameters
contextUse Context with a Sub-Timeline to specify whether the returned value is relative to the Sub-Timeline or the main Timeline. If the Timeline is not a Sub-Timeline, the context uses local time regardless of the specified context.

Use Context.Local, the default, to retrieve the playhead location relative to the Sub-Timeline. Use Context.Global to retrive the playhead location relative to the main Timeline.

Exceptions
System.InvalidOperationExceptionThe Window associated with this instance has been destroyed.
System.ArgumentExceptionThe context is invalid.
Returns
The playhead location in frames.
