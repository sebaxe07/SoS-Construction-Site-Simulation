void Start ()
 Initializes the UIManager by setting up the ProjectManager instance, initializing data structures, adding button listeners, and displaying saved constellations.
 
void LoadWorkers ()
 Loads worker data from JSON file into the available workers list. Only includes workers that aren't already in the selected workers list. Clears the available workers list before loading new data.
 
void CreateConstellationPage ()
 Creates and displays a new constellation creation page. Clears previous selections, instantiates the popup UI, sets up UI elements, and configures button listeners for saving and canceling. Also initializes the machine/worker toggle functionality.
 
void GetAvailableWorkers ()
 Retrieves the list of available workers by loading workers from storage and filtering out any workers that are already selected.
 
void UpdateAvailableWorkerList ()
 Refreshes the UI display of available workers. Clears existing worker UI elements and creates new ones for each available worker. Includes worker details like ID, surname, role, and status.
 
void UpdateMachineList ()
 Updates the UI display of available machines. Clears existing machine UI elements, loads machines from the ProjectManager, and displays them organized by machine type with relevant details and selection buttons.
 
void UpdateSelectedConstellationList ()
 Updates the UI display of selected workers and machines in the constellation. Filters out workers already assigned to machines and displays both unassigned workers and machines with their assigned workers if applicable.
 
void SelectWorker (Worker worker)
 Handles the selection of a worker by moving them from available to selected list and updates both the available and selected worker displays.
 
void SelectMachine (Machine machine)
 Handles the selection of a machine by moving it from available to selected list, organizing it by machine type, and updating the UI displays.
 
void RemoveWorker (Worker worker)
 Removes a worker from the selected list and adds them back to available workers. Updates both the available and selected worker displays.
 
void RemoveMachine (Machine machine)
 Removes a machine from the selected list and adds it back to available machines. Updates both the available and selected machine displays.
 
void DisplaySavedConstellations ()
 Loads and displays all saved constellations in the UI. Creates UI elements for each constellation with options to delete or edit.
 
void DeleteConstellation (HelperConstellations.SavedConstellation constellation)
 Deletes a saved constellation from both the list and storage file. Updates the UI display after deletion.
 
void EditConstellation (HelperConstellations.SavedConstellation constellation)
 Opens the constellation editor with pre-populated data from an existing constellation. Loads associated workers and machines, sets up their assignments, and configures the save button to update instead of create new.
 
void UpdateConstellation (string constellationId)
 Updates an existing constellation with new data. Validates the constellation name, checks for duplicates, updates worker and machine assignments, and saves changes to storage.
 
void AssignWorker (Machine machine)
 Creates a popup interface for assigning a worker to a specific machine. Displays available workers and handles the assignment process, including updating worker lists and machine assignments.
 
void UpdateEveryList ()
 Updates all UI lists based on the machine/worker toggle state. Refreshes either the machine list or worker list along with the selected constellation list.
