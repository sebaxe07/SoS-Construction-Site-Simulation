Detailed Description

An interface that represents a brush's controller.

This interface is used for implementing custom controllers for handling brush properties.

See also
IBrushStrengthController, IBrushSizeController, IBrushRotationController, IBrushScatterController, IBrushSpacingController, IBrushSmoothController, IBrushModifierKeyController
