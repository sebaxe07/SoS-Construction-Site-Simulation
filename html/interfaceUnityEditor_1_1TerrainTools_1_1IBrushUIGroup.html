◆ ScatterBrushStamp()

bool UnityEditor.TerrainTools.IBrushUIGroup.ScatterBrushStamp ( ref Terrain terrain,
ref Vector2 uv )

Scatters the brush around the specified UV on the specified terrain. If the scattered UV leaves the current terrain then the terrain AND UV are modified for the terrain the UV is now over.

Parameters
terrainThe terrain the scattered UV co-ordinate is actually on.
uvThe UV co-ordinate passed in transformed into the UV co-ordinate relative to the scattered terrain.
Returns
"true" if we scattered to a terrain, "false" if we fell off ALL terrains.
