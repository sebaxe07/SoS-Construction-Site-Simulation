Installation

Use the Package Manager to install the Post Processing package or to update an installed package to the latest version.

To install the Post Processing package:

  1. Open the Package Manager in Window > Package Manager.
  2. In the Filter for scope menu, select All packages.
  3. In the list of packages, select Post Processing.
  4. In the bottom-right corner of the Package Manager window, select Install.
