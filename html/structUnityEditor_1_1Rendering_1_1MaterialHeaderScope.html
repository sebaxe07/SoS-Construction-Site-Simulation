Detailed Description

Create a toggleable header for material UI, must be used within a scope.

Example:

void OnGUI()
{
using (var header = new MaterialHeaderScope(text, ExpandBit, editor))
{
if (header.expanded)
EditorGUILayout.LabelField("Hello World !");
}
}
